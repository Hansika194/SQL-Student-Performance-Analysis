# SQL-Student-Performance-Analysis
SQL Data Analysis Internship –  Advanced SQL using JOIN, GROUP BY, HAVING, Subqueries, Student Performance Analysis.

Project Overview

This project is part of my SQL Data Analysis Internship. The objective is to analyze student performance using advanced SQL concepts such as JOINs, GROUP BY, HAVING, aggregate functions, and subqueries.

The project demonstrates how SQL can be used to extract meaningful insights from a relational database.

---

## 🎯 Objectives

- Find the top student in each course.
- Calculate the pass rate for every course.
- Identify the overall topper across all courses.
- Find students enrolled in multiple courses.
- Practice advanced SQL queries for real-world data analysis.

---

## 🗂 Database Structure

The project uses three related tables:

- **Students** – Stores student information.
- **Courses** – Stores course details.
- **Enrollments** – Connects students with courses and stores grades.
